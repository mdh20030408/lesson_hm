设计一个支持push，pop，top等操作，并能**常数时间O(1)**检索到最小元素的栈

- 栈是什么？
  - FIFO 逻辑 一端操作
  - 线性数据结构 (受限的)
  - ADT  抽象数据类型
    存储结构 + 行为
    - 数组 简单 + 连续  new Array(10).fill(0)
    - 链表 复杂 + 指针的开销 不连续 执行(new)

    - 栈的行为
      - push  入栈
      - pop  出栈
      - top  查看栈顶元素
      - size 查看栈的大小
      - isEmpty 查看栈是否为空
      - clear 清空栈
